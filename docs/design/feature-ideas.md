# 创意生成平台探索型 Feature 列表

> 个人练手用，聚焦有技术含量的玩法，可选组合，不必一次全部做。默认假设无鉴权。

## 1) 创意生成与生产力
- **多尺寸/多模态一键出图**：同一任务同时生成 1:1、9:16、16:9，支持上传参考图 + 品牌 Logo 叠加。技术点：任务拆分与聚合、生成尺寸到格式映射、图片合成流水线。
- **版式模板引擎**：定义文字/Logo/CTA 占位的模板 JSON，结合生成图片进行二次排版输出成最终广告图。技术点：自定义模板 DSL、节点约束（安全区、对齐）、Canvas/Go 图像处理。
- **批量生成管线**：CSV/Excel 导入商品列表自动创建任务，进度仪表盘实时反馈。技术点：大批量任务队列、分批入库、流式上传/解析。
- **Prompt 版本管理**：同一商品多套提示词版本，记录模型参数和效果评分，便于回溯。技术点：元数据模型、差异视图、版本快照。

## 2) 质量与安全
- **多模型评分排序**：生成后调用 CLIP/Aesthetic/NSFW/简易 CTR 预测模型写入 `creative_scores`，列表按综合得分排序。技术点：批量离线推理、GPU/CPU 混合调度、结果归一化和权重配置。
- **自动裁切与抠图**：对生成图片做主体检测+抠图，再匹配版式模板自动摆放。技术点：开源分割模型调用、掩码合成、画布自适应布局算法。
- **水印/版权标记**：输出图自动叠加半透明水印与生成信息（模型、时间、task_id），可配置位置与透明度。技术点：图像叠加、配置化流水线。

## 3) 实验与分析
- **A/B 试验框架**：同一商品多变体，自动分组记录曝光/点击（可用假数据或脚本模拟），计算转化率和显著性。技术点：试验模型、指标收集接口、简单显著性检验。
- **多维筛选与标签系统**：给素材打标签（风格、行业、颜色、场景），支持组合过滤和收藏夹。技术点：标签多对多表、查询索引、前端多选筛选 UI。
- **可视化报告**：任务/素材时序、尺寸分布、模型版本对比图表。技术点：SQL 统计、后端聚合接口、前端图表库（ECharts/Recharts）。

## 4) 体验与实时性
- **WebSocket/SSE 推送**：任务状态、评分完成、上传结果实时推送，前端状态同步。技术点：连接管理、断线重连、前端 store 更新。
- **可中止/重试任务**：为长耗时生成提供 cancel/retry，队列层面优先级调整。技术点：任务状态机、幂等处理、后台 worker 协议。
- **离线批处理模式**：提供 CLI 或 cron 定时批量生成，结果落盘+通知。技术点：命令行入口、批处理日志、结果归档目录结构。

## 5) 集成与扩展
- **存储抽象切换**：在 MinIO/七牛/本地之间切换，提供预签名 URL 与 CDN 域名管理。技术点：存储 provider 接口、配置热加载、URL 策略。
- **模型路由与降级**：通义不可用时自动切到本地/其他模型，或降级到低分辨率。技术点：健康检查、路由策略、超时与重试。
- **导出/打包**：一键导出某任务的所有素材+元数据（JSON/CSV）和小预览页，便于分享。技术点：打包压缩、预览 HTML 生成、短链（本地映射即可）。

## 6) 前端交互增强
- **三步创意工作流**：商品→文案候选→图像生成的向导式 UI，支持候选编辑和格式预览。技术点：前端状态机、乐观更新、错误恢复。
- **模板可视化编辑器（简版）**：拖拽文字/Logo/CTA 占位，保存为模板 JSON 并用于自动排版。技术点：Canvas/SVG 编辑、对齐与吸附、模板持久化。
- **快捷操作**：素材卡片直接“重新生成同款/放大/调色”，触发后台二次处理。技术点：复用任务参数、二次处理流水线。

> 选 2-3 个你最感兴趣的先做：推荐从“多尺寸+模板排版”、“评分排序”、“WebSocket 实时推送”或“文案工作流”开始，既有技术深度又能看到明确成效。
